Accountability: About This Data Set

The Winter Olympics data set compiled by The Guardian and posted publicly to Kaggle is accountable and easily fact-checked by any number of external websites and resources that contain the indentical information presented in the data.

Where did this data come from?

This data was compiled by the British media outlet, The Guardian and posted to Kaggle, a free website which hosts data science collaborations and competitions. The data on the Winter Olympics contains all 5770 medal winners dating back to the first Winter Olympic Games in 1924 up until the most recently completed games in Sochi, Russia in 2014. Furthermore, the data has nine distinct fields:

  • Year
  • City (Host)
  • Sport
  • Discipline
  • Athlete
  • Country
  • Gender
  • Event
While this data exists in other formats around the internet in pieces, this particular data set as compiled by The Guardian is more comprehensive and clearly underwent some degree of fact-checking and data cleaning prior to its upload to Kaggle.

Who has access to it?

In line with the true spirit of the Olympics, this data set is publicly available and can be downloaded and manipulated free of charge. To access the data, one can simply visit Kaggle and select the Download option.
